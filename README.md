# TicTacToe

This is a tic-tac-toe program

## How to play

In order to play:\
\
  1. Run the program:\
        `ruby main.rb`\
  2. Input the first player's name (X)\
  3. Input the second player's name (O)\
  4. In order to select a square type in it's column and then row separated by a '-':\
        `1-3`
